# Frontend - React App

Cette application web React pour gérer les finances personnelles.

## Installation

```bash
cd frontend
npm install
npm start
```

## Structure

- `src/pages/` - Pages principales de l'application
- `src/components/` - Composants réutilisables
- `src/services/` - Services API
- `src/styles/` - Fichiers CSS

## Fonctionnalités

✅ Tableau de bord avec résumé financier
✅ Gestion des revenus
✅ Gestion des dépenses
✅ Gestion de l'épargne
✅ Gestion des assurances
✅ Rapports et graphiques
✅ Paramètres utilisateur

## Connexion API

L'application se connecte au serveur backend sur `http://localhost:5000`
